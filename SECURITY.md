## 报告漏洞

Mem Reduct 使用 Native API 执行内存清理，需管理员权限。若发现**安全相关**问题：

1. **请勿**在公开 Issue 中披露可利用细节
2. 可通过 [GitHub Security Advisories](https://github.com/gradient30/memreduct/security/advisories/new) 私下报告（若仓库已启用）
3. 或联系仓库维护者（见 GitHub 个人资料）

上游 Henry++ 原版的安全问题也可向 [henrypp/memreduct](https://github.com/henrypp/memreduct) 反馈。

## 说明
